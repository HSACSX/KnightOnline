Detailed Description

Represents a single sound object, either buffered or streamed.

CN3SndObj manages a single sound instance, including playback, looping, volume, 3D position, and lifetime of the underlying audio asset. It interacts with the sound manager to queue OpenAL operations on a dedicated audio thread.
